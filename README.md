재생 플라스틱 섬유

재활용 쓰레기 플라스틱 그리고 바다,산 등 다양한 곳에서 
발견되는 플라스틱쓰레기들을 재생 플라스틱 섬유라는 옷, 신발 등에서

해양 플라스틱 쓰레기를 통해 운동화 제작 협업을 하기도 하면서, 이와같이 플라스틱 쓰레기 재활용의 특별함을 보여주고 있다
